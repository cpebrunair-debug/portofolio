Dashboard Model

Alur model dashboard BCRIP

Dashboard ini dirancang untuk membaca data aset, menghubungkannya dengan paparan multi-bahaya, lalu menerjemahkannya menjadi prioritas risiko dan estimasi dampak finansial yang lebih mudah dibaca oleh pengguna operasional maupun pengambil keputusan.

01
Input aset
Pengguna memasukkan identitas aset, lokasi, nilai aset, dan kategori industri sebagai basis analisis.
02
Ekstraksi bahaya
Sistem membaca layer geospasial multi-bahaya untuk menghasilkan indeks risiko banjir, gempa, longsor, dan kekeringan.
03
Kuantifikasi finansial
Nilai aset dan level risiko diolah menjadi paparan finansial, value at risk, serta ringkasan portofolio yang lebih terukur.
04
Prioritas dashboard
Hasil analisis disajikan ke dalam dashboard, peta, tabel aset, dan rekomendasi mitigasi untuk mendukung pengambilan keputusan.

Cara menggunakan form input aset

Cara Mengisi Form

1. Cari Lokasi — Ketik nama lokasi atau alamat, lalu pilih dari dropdown hasil pencarian. Koordinat latitude/longitude akan terisi otomatis.

2. Klik Peta — Alternatifnya, klik langsung pada peta interaktif untuk menentukan posisi aset.

3. Skor Bahaya — Setelah lokasi dipilih, sistem akan otomatis mengquery data COG via TiTiler untuk menghasilkan indeks risiko (0–1) untuk setiap jenis bencana.

4. Simpan — Klik "Simpan Aset" untuk menyimpan data ke penyimpanan lokal browser. Data disimpan di localStorage dan tetap tersedia hingga cache browser dihapus.

5. Export/Import — Tekan Ctrl+Shift+D untuk mengakses tools ekspor/impor data JSON sebagai backup.

📖 Metodologi Penilaian Risiko

Indeks 0–1: Semua nilai bahaya dinormalisasi ke skala 0–1. Nilai 0 = tidak ada paparan, nilai 1 = paparan maksimum.

Status At Risk: Aset dikategorikan At Risk jika indeks bahaya tertinggi (max dari 4 bahaya) ≥ 0.34 (level Sedang atau Tinggi).

Value at Risk (VaR): Total nilai finansial aset yang berada di kategori At Risk.

Estimasi Paparan Finansial: Perkiraan potensi kerugian berdasarkan level risiko:

LevelEstimasi Dampak
Rendah5% dari nilai aset
Sedang20% dari nilai aset
Tinggi50% dari nilai aset
